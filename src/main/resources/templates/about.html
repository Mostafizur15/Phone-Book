About Contact Management System

This Contact Management System is a web-based application designed to help users efficiently manage their personal and professional contacts. With a clean and user-friendly interface, it allows you to store, view, edit, and share contact information securely.

Features

  • Create and manage contact details with profile pictures
  • Search and filter contacts easily
  • Send selected contacts via email
  • Role-based access (optional)
  • Mobile responsive design

Technologies Used

  • Spring Boot 3 (Backend)
  • Thymeleaf (View Engine)
  • Java 17
  • MySQL (Database)
  • HTML, CSS, JavaScript, Bootstrap (Frontend)
